# A REACT-NATIVE APP TEMPLATE FOR MYSELF

## features enabled

1. Typescript 2.0
2. Yarn
3. Typings
4. Gulp
5. TSLint
6. Editorconfig
7. Immutable
8. Redux
9. Redux-UI

## install

1. `react-native init MY_AWESOME_PROJECT`
2. `cd MY_AWESOME_PROJECT`
3. `git remote add template GIT_REPO_OF_THIS_PROJECT`
4. `git pull template master`
5. `yarn install`
6. `typings install`
7. `gulp watch`
8. `react-native run-ios`
